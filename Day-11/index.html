There are three ways which helps html to connect with css.

  1. Inline Css
  2. Internal Css
  3. External Css

Inline Css

Inline Css is a type of css using inside a tag.Example- 


<p style="background-color: red; color:yellow"></p>

Internal Css

Internal Css is a type of css using below title style tag.Example,in this also use the below code.- 

<style>
div{
    background-color: rgb(88, 250, 101);
} 
<style/>

External Css

External Css is also a type of css in which using inside html file we put link tag which connect to external file .css. Example here- 

<link rel="stylesheet" href="style.css">
